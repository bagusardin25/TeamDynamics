# TeamDynamics

A multi-agent AI sandbox to explore office politics, morale, and productivity.

## How to Run the Project (Frontend Only)

Currently, the frontend UI is built. Here is how you run it locally:

1. Open a terminal.
2. Navigate into the website folder:
   ```bash
   cd "Prepare Hackathon S2/TeamDynamics/website"
   ```
3. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to: [http://localhost:3000](http://localhost:3000)

*(Note: The Backend FastAPI engine has not been built yet. The current frontend is running with mock visual data to demonstrate the intended layout and experience UX).*
